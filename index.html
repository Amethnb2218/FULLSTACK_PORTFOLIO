2. HTML

1) Présentation de HTML

HTML (HyperText Markup Language) est le langage de balisage qui sert à structurer le contenu d'une page Web: titres, paragraphes, listes, liens, images, formulaires, etc.

2) Concepts

  • a) Élément HTML: unité complète formée d'une balise ouvrante, du contenu, puis d'une balise fermante.
  • b) Balise HTML: mot-clé entre chevrons, par exemple <p> ou <img>.
  • c) Attribut HTML: information complémentaire dans la balise, par exemple href, src, alt, class.
  • d) Structure d'une page HTML: doctype, html, head, body et contenu sémantique.
  • e) Publier du texte: utilisation des titres (<h1>...<h6>) et paragraphes (<p>).
  • f) Lister du contenu: listes ordonnées (<ol>) et non ordonnées (<ul>).
  • g) Lier des contenus: ancres (<a>) pour relier des pages et sections.
  • h) Afficher une image: balise <img> avec src et alt.
  • i) Présenter un formulaire: balise <form> avec champs input, textarea, select, button.

3. CSS

1) Présentation du CSS

CSS (Cascading Style Sheets) permet de styliser la structure HTML: couleurs, espacements, tailles, alignements, animations et mise en page responsive.

2) Concepts

  • a) Sélecteur CSS: cible les éléments à styliser.
  • b) Propriété CSS: caractéristique visuelle à modifier (color, margin, display...).
  • c) Valeur CSS: valeur associée à une propriété (red, 1rem, flex...).
  • d) Règle CSS: combinaison sélecteur + bloc de déclarations.
  • e) Application du CSS: inline, interne ou externe.
  • f) Sélecteur de type: cible une balise (p, h1, img...).
  • g) Sélecteur d'attribut: cible selon un attribut ([type="text"]).
  • h) Sélecteur d'identificateur: cible un id unique (#menu).
  • i) Sélecteur de classe: cible une classe réutilisable (.carte).
  • j) Combinateurs: relation entre éléments (descendant, enfant direct, adjacent...).
  • k) Modèle de boîte: content, padding, border, margin.
  • l) Boîtes flexibles: mise en page flexible avec Flexbox.
