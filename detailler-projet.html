Détails du projet: Jolofera

  • Libellé: Jolofera - Plateforme SaaS Réservation & E-commerce
  • Description:

    Jolofera est une plateforme SaaS multi-tenant dédiée aux salons de beauté, coiffeurs et boutiques. Le projet permet la réservation de prestations en temps réel, la vente de produits, le paiement intégré multi-passerelles, ainsi que la gestion de médias et notifications pour une expérience client complète.

  • Technologies utilisées:
    1. React 18 + Vite + Tailwind CSS
    2. Node.js + Express.js
    3. PostgreSQL + Prisma ORM
    4. WebSocket (temps réel)
    5. Cloudinary + Google Auth
  • Rôle: Fondateur et Développeur Full Stack
  • Période: 2024 - Présent
